### Galvenās funkcijas

* **Pārvilkšanas sistēma :** Realizēta pilnvērtīga bruņu uzlikšanas loģika. Priekšmetus var pārvilkt no inventāra uz attiecīgajām vietām (ķivere, krūšu bruņas, bikses, zābaki).
* **Viedie sloti:** Katrs slots pieņem tikai atbilstoša tipa bruņas un neļauj vienā vietā novietot vairākus priekšmetus vienlaicīgi.
* **Dinamiskā mērogošana (Scale System):**
    * Slīdnis **Growth**: maina varoņa augumu.
    * Slīdnis **Width**: maina varoņa platumu.
* **Varoņa izskatu (Skin) maiņa:** Integrēta izvēlne (Dropdown), lai pārslēgtos starp dažādiem tēliem (Steve, Alex u.c.) ar dinamisku apraksta atjaunošanu.
* **Bruņu vizualizācija:** Tiklīdz ikona tiek ievietota slotā, uz 2D varoņa modeļa automātiski parādās atbilstošais bruņu elements (ādas, dzelzs, zelta vai dimanta).
